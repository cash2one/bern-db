Introduction

What is Bern DB?

Bern DB is an API with quotes from Bernie Sanders.

API

General

All endpoints share this structure:
https://api.bern-db.com/v0/

Rate Limiting

A maximum of 100 requests per minute and 1000 requests per day are allowed.

The response headers indicate your current rate limit status. 

HTTP/1.1 200 OK
Server: nginx/1.4.6 (Ubuntu)
Date: Wed, 24 Feb 2016 16:51:46 GMT
Content-Type: application/json
Content-Length: 458
Connection: keep-alive
Access-Control-Allow-Origin: *
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 97
X-RateLimit-Reset: 1456332739
X-UA-Compatible: IE=Edge

Quotes

Quotes are represented as an object with the following structure. 

{
  "author": "Bernie Sanders", 
  "credit": "PBS NewsHour", 
  "credit_url": "http://www.pbs.org/newshour", 
  "date": "2016-02-11", 
  "id": 2, 
  "source_url": "http://newshour-tc.pbs.org/newshour/wp-content/uploads/2016/02/PBS-NewsHour-Dem-Debate-%E2%80%93-FULL-Rush-Transcript.pdf", 
  "tags": [
    "government"
  ], 
  "text": "I believe in government, but I believe in efficient government, not wasteful government.", 
  "uri": "http://api.bern-db.com/v0/quotes/2"
}
